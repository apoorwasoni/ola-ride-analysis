# Ola Ride Trend Analysis

## Project Overview
This project analyzes Ola ride booking data to understand ride trends, cancellation patterns, and customer behavior. The analysis helps identify key insights to support business decisions and improve operations.

## Dataset
- **File:** (https://github.com/apoorwasoni/ola-ridetrend-analysis/blob/main/Bookings-20000-Rows.xlsx%20-%20July.csv)
- **Rows:** 20,000
- **Columns:** ride_id, date, pickup_location, drop_location, fare, distance, status, payment_mode

## Key Objectives
- Analyze daily and weekly ride trends
- Study cancellation patterns and identify peak times
- Segment rides/customers based on usage patterns
- Provide actionable insights via dashboards

## Dashboard
- A PDF dashboard (`Ola dashboard.pdf`) summarizes the key insights with visualizations.

## Tools Used
- Excel
- SQL
- Power BI

## How to Use
1. Download the dataset from this repository.
2. Explore and analyze the data using Python or SQL.
3. View the dashboard PDF for summarized insights.
